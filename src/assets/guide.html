Системные требования

  • Наличие подключения к сети Интернет;
  • Тип операционной системы – UNIX-подобные системы, Windows 7 и выше;
  • Браузер – Google Chrome 86.0.4240.183 (64-битный) и выше, Яндекс.Браузер 19.1 и выше.
  • Манипуляторы: мышь, клавиатура

Введение

В системе присутствует две роли: администратор и обучающийся. Администратор может создавать новые упражнения, настраивать уровень сложности, просматривать статистику по обучающимся и упражнениям. Обучающийся выбирает уровень сложности и выполняет соответствующие упражнения, после выполнения обучающийся может оценить собственную статистику.

Авторизация

Для регистрации в системе необходимо ввести логин, email, пароль. Для создания аккаунта пользователю необходимо нажать на кнопку "Зарегистироваться".

Для входа в систему вам необходимо ввести логин и пароль и нажать кнопку "Войти".

Режим "Администратор"

После авторизации под ролью "Администратор", интерфейс будет подстроен под роль. Перед вами будет страница с созданными упражнениями.

Настройка уровня сложности

Для настройки уровня сложности необходимо нажать на кнопку "Настроить уровень сложности". После этого откроется соответствующая страница.

На странице представлен список уровней сложности: уровень 1, уровень 2, уровень 3. Для настройки уровня сложности необходимо задать такие параметры, как диапазон времени выполнения упражнения, диапазон длины упражнения, зону работы с клавиатурой, а также максимальное количество ошибок. Для сохранения опций необходимо нажать на кнопку "Настроить".

Создание упражнения

Для перехода к созданию упражнения, нажмите на кнопку "Создать новое упражнение" на странице упражнений. На данной странице необходимо задать такие параметры, как название упражнения, уровень сложности, время выполнения, длина упражнениия, а также выбрать способ генерации текста упражнения.

Способ генерации текста упражнения

Текст упражнения можно создать двумя способами: ручным и автоматическим. При выборе ручного способа создания необходимо ввести текст упражнения в соответствующее поле. При автоматической генерации текста, система сгенерирует текст упражнения соответственно заданным опциям.

Просмотр статистики

Для просмотра статистики администратору необходимо нажать на соответствующую кнопку на навигационной панели. Данная страница разделена на 2 блока: статистика по упражнениям и статистика по обучающимся. Администратор выбирает тип интересующей статистики на навигационной модели. При выборе статистики по упражнениям будет выведен список всех упражнений с описанием их параметров, для того чтобы визуализировать статистику необходимо выбрать упражнение из списка. Статистика по обучающимся представлена аналогичным образом.

Режим "Обучающийся"

После авторизации под ролью "Обучающийся", интерфейс будет подстроен под роль. Перед вами будет страница со списком упражнений.

Выбор уровня сложности

Для того чтобы подобрать упражнения, пользователю небходимо выбрать уровень сложности в выпадающем списке.

Просмотр списка упражнений

После выбора уровня сложности система сгенерирует список соответствующих упражнений.

Выбор упражнения для выполнения

Для того чтобы посмотреть текст упражнения необходимо нажать на кнопку "Показать текст". Для выполнения упражнения обучающийся должен нажать на кнопку "Выполнить".

Выполнение упражнения

При нажатии на кнопку "Выполнить упражнение" открывается страница выполнения упражнения, на которой представлен текст упражнения, счетчик времени, количество ошибок, поле ввода.

Подключение виртуальной клавиатуры

Для подключения виртуальной клавиатуры необходимо нажать на кнопку "Подключить виртуальную клавиатуру". Если виртуальная клавиатура подключена, то ее можно скрыть,нажав на кнопку "Скрыть виртуальную клавиатуру".

Ввод текста упражнения

Текст упражнения печатается в специальную форму ввода. При вводе символа система сравнивает текущий символ с символом в тексте упражнения, при правильном вводе символ подсвечивается синим, при неправильном символ в тексте становится красным. Для продолжения выполнения упражнения необходимо исправить ошибку и ввести правильный символ

Результат выполнения упражнения

Когда упражнение будет полностью выполнено или закончится время выполнения упражнения, система откроет страницу результата выполнения упражнения, на которой представлено время выполнения упражнения, средняя скорость набора,количество допущенных ошибок,а также кнопка "Выполнить еще раз" для повторного выполнения упражнения.

Просмотр собственной статистики

Для того чтобы посмотреть статистику выполнения упражнений, пользователю необходимо открыть страницу "Статистика" на навигационной панели. На данной странице представлен список выполненных упражнений, для того чтобы оценить результат выполнения, необходимо выбрать упражнение из списка, справа от него появится диаграмма, отражающая собранную статистику.
